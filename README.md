# AI Job Market Analyzer

## Overview
This project analyzes LinkedIn job postings to identify hiring trends, in-demand skills, top employers, and geographic job distribution in the data analyst market.

It combines Python-based data analysis with AI-generated insights using Google Gemini.

---

## Problem Statement
Understanding what skills, roles, and locations dominate the data analyst job market helps job seekers and professionals make informed career decisions.

---

## Dataset
- 2,337 LinkedIn job postings
- Includes job titles, companies, skills, and locations

---

## Tech Stack
- Python
- Pandas
- Matplotlib
- Google Gemini AI (Generative AI API)

---

## Project Workflow

1. Data Cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Skill frequency analysis
4. Company and location distribution analysis
5. Visualization using Matplotlib
6. AI-generated insights using Gemini API

---

## Key Insights

- Data Analyst roles dominate the job market (~1147 listings)
- Increasing demand for specialized roles (Finance, Supply Chain, Growth Analytics)
- Strong geographic concentration in South Africa (Johannesburg, Cape Town)
- Core skills: SQL, Excel, Power BI, Tableau, Python
- Machine Learning is still a niche requirement for most analyst roles
- Growing importance of data quality and governance roles

---

## AI Component

Google Gemini was used to:
- Summarize market trends
- Generate business insights
- Provide career recommendations for freshers

---

- Extracted top skills and employer patterns

- Built an end-to-end data analytics pipeline with AI integration
